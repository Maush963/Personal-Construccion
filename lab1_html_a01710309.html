Preguntas y Respuestas sobre HTML5

  • ¿Cuál es la diferencia entre Internet y la World Wild Web? La diferencia es que Internet es una red de redes, mientras que la World Wide Web es un sistema de información que se encuentra en Internet.

  • ¿Cuáles son las partes de un URL? Esquema, nombre de dominio, ruta, parámetros y fragmento.

  • ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

    GET Recuperar datos del servidor. Es idempotente
    HEAD Similar a GET, pero solo solicita encabezados, sin cuerpo de respuesta. Verificar existencia de un recurso
    POST Enviar datos al servidor para ser procesados y almacenados. No es idempotente
    PUT Reemplazar todos los datos de un recurso en el servidor.
    PATCH Actualizar parcialmente un recurso en el servidor.
    DELETE Eliminar un recurso en el servidor.

  • ¿Qué método de HTTP se debe utilizar al enviar un formulario HTML (Para contraseñas)? POST porque lo encripta, no va en el URL.

  • ¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL? GET porque estás solicitando al servidor que te envíe los datos de la página web especificada por el URL.

  • Un código 200 ¿qué significa? Indica que la solicitud ha tenido éxito.

  • Error 404 Indica que la página no se encontró.

  • Error 505 "Versión de HTTP no soportada".

  • Menciona algunos elementos de HTML 4 que en HTML 5 estén desaprobados: "acronym", "applet", "big".

  • ¿Cuáles son las principales diferencias entre HTML 4 y HTML 5? HTML5 tiene nuevos elementos semánticos que facilitan la estructuración y comprensión del contenido de una página web.

  • ¿Qué componentes de estructura y estilo tiene una tabla? "Table" para tabla, "tr" para tabla, "td" para celda de datos en la tabla, "th" Define un encabezado de columna o fila en la tabla.

Preguntas y Respuestas sobre CSS

  1. Como ingeniero de software ¿Cuál es tu recomendación sobre el uso de !important en un CSS?

    No usarlo excesivamente ya que sobrescribe el estilo previamente definido y puede causar conflictos.

  2. Si se pone una imagen de fondo en una página HTML, ¿Por qué debe escogerse con cuidado?

    Debe tener una resolución adecuada para que se vea bien en varios dispositivos y no debe ser muy pesado.

  3. Como ingeniero de software ¿Cuál es tu recomendación al elegir las unidades de una propiedad de estilo entre %, px y pt?

    Tienen usos distintos: "%" es para diseños responsivos y fluidos, "px" ofrece control preciso y "pt" es para impresiones.

  4. ¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

    Hace que el archivo enviado al navegador sea más pequeño, lo que resulta en un tiempo de carga más rápido.

    Preguntas a Responder

  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    Su principal diferencia reside en el propósito para el que operan, Java es utilizado para desarrollo de aplicaciones, mientras JavaScript es usado principalmente para desarrollo web. Ambos tienen Bibliotecas y marcos de trabajo específicos para sus áreas.

  • ¿Qué métodos tiene el objeto Date?

    getDate(), Devuelve el día de mes, getMonth(), Devuelve el mes, getFullYear(),Devuelve el año, getDay, devuelve el día de la semana *lunes*, toDateString(), Devuelve la parte de la fecha de un objeto Date como una cadena legible.

  • ¿Qué métodos tienen los arreglos?

    Push(), agrega un o más elementos, Pop(), elimina el último elemento, Splice(), reemplaza elementos del arreglo, forEach(), Ejecuta una función proporcionada una vez por cada elemento del arreglo, indexOf(), devuelve el índice del primer elemento en el arreglo que coincida con el valor especificado.

  • ¿Cómo se declara una variable con alcance local dentro de una función?

    Simplemente se utiliza "var" o "const" dentro del cuerpo de una función.

  • ¿Qué implicaciones tiene que utilizar variables globales dentro de funciones?

    Además de afectar el rendimiento del código hace que sea más difícil de entender el flujo de datos que tiene el programa.
